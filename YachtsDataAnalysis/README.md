# Krakow Apartment Offerts
Kacper Osmola 2026

It's a quick analysis of the zegluj.pl yachts offerts for renting on some may weekend. The data is webscrapped and
then analyzed. This project was for the university, so it is short. 

This project contains:
- Web scraping
- Data cleaning and preparation
- Data visualization with different types of graphs including maps

Technologies I used:
- Python
- Pandas
- Matplotlib and seaborn
- Data visualization
- Requests and beautifulsoup
