# Krakow Apartment Offerts
Kacper Osmola 2026

It's a quick analysis of the Otomoto apartment offerts in Poland. I made this project to train my data visualization
skills and check new kinds of graphs

Source: https://www.kaggle.com/datasets/amrahhasanov23/otodom-pl-flat-prices-in-poland

This project contains:
- Data cleaning and preparation
- Data visualization with different types of graphs including maps

Technologies I used:
- Python
- Pandas
- Matplotlib and seaborn
- Data visualization
