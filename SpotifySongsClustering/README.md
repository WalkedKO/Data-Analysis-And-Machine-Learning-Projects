# Spotify Songs Clustering
Kacper Osmola 2026

We have a dataset of plenty of songs from Spotify. Can we build a system, which will cluster them into some playlists?
The data set is from: https://www.kaggle.com/datasets/rishabhpancholi1302/spotify-most-popular-songs-dataset/code

This project contains:
- Data cleaning and preparation
- Brief analysis and data visualization
- Lyrics analysis
- Retrieval, trying to find most similar songs to a given one based on track attributes and lyrics 
- Clustering based on track attributes and lyrics using:
	- K-means
	- Agglomerative clustering 
	- DBSCAN

Technologies I used:
- Python
- Pandas
- Matplotlib and seaborn
- Data visualization
- Clustering and Retrieval
- Text document representation
- K-means, Agglomerative clustering and DBSCAN
